# 🧠 또바 (TtoBa) – 웹 기반 AI 모의면접 플랫폼

> **“누구나 실제 면접처럼 연습하고, AI가 나의 표정·시선·답변을 분석해주는 개인 맞춤형 모의면접 서비스”**

---

## 🎯 프로젝트 개요
‘또바’는 브라우저에서 카메라와 마이크를 켜고 실제 면접처럼 답변하면,  
**AI가 실시간으로 표정·시선·음성·논리 등을 분석해 리포트를 제공**하는  
**웹 기반 B2C AI 모의면접 플랫폼**입니다.

- **목표**: 개인의 면접 역량 향상 및 자기 피드백 지원  
- **문제 인식**: 기존 서비스는 B2B 중심, 고비용 구조로 개인 접근성이 낮음  
- **핵심 가치**: *AI 기반 개인 성장 중심 학습 플랫폼*

---

## 🌟 주요 기능
✅ **직무별 맞춤 면접 시나리오**  
개발·마케팅·디자인 등 직무별 질문과 평가 기준 자동 구성  

✅ **AI 실시간 분석**  
- Whisper로 음성 → 텍스트 변환 (STT)  
- 답변 시간·속도·언어 습관 기록  

✅ **AI 리포트 자동 생성**  
- 평가 항목: 논리성 · 전문성 · 어휘력 · 발음 · 태도  
- 시각화: 점수 그래프 + 개선 방향 제시  

✅ **성장 대시보드**  
- 이전 면접 기록과 비교  
- 개인별 발전 추세 확인 가능  

---

## ⚙️ 기술 구조
| 구분 | 기술 스택 |
|------|------------|
| **Frontend** | React, WebRTC |
| **Backend** | FastAPI |
| **AI 모델** | Whisper(STT), GPT-4o mini(답변 분석), MediaPipe(시선·표정 인식) |
| **Database** | SQLite (영상은 임시 저장 후 자동 삭제) |
| **Visualization** | JSON 기반 그래프 시각화 |

## 🚀 기대 효과
- 실제 면접에 가까운 **몰입형 연습 경험 제공**  
- AI 리포트를 통한 **객관적 자기 분석 및 개선 방향 도출**  
- **성장 데이터 시각화**로 자기 발전 확인  
- 대학 취업지원센터·HRD 분야 **확장 가능성**  
- 향후 **AI 코칭·멘토링 결합 서비스**로 고도화 가능
