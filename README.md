# 🧠 또바 (TtoBa) – 웹 기반 AI 모의면접 플랫폼

‘또바’는 누구나 브라우저에서 실제 면접처럼 카메라와 마이크를 켜고 답변하면, **AI가 표정·시선·음성·답변 논리 등을 실시간 분석**해 **맞춤형 리포트**를 제공하는 웹 기반 **B2C AI 모의면접 플랫폼**이다.

---

## 🎯 프로젝트 개요
- **목표**: 개인의 면접 역량 향상과 자기 피드백에 초점을 맞춘 AI 모의면접 서비스
- **문제 인식**: 기존 서비스는 B2B 중심·고비용 구조로 개인 접근성이 낮음
- **핵심 가치**: AI 분석 기반의 *개인 성장 중심 피드백* 제공

---

## 🧩 주요 기능
- 직무별(개발/마케팅/디자인 등) 맞춤 **질문 및 평가 기준**
- **실시간 STT**로 답변 텍스트화 및 언어 습관 기록
- 면접 종료 후 **AI 리포트 자동 생성**
  - 논리성, 전문성, 어휘력, 발음, 태도 등 점수화
  - 개선 방향 제시 및 성장 추이 시각화
- **대시보드**를 통해 연습 기록 및 발전 추세 확인

---

## ⚙️ 기술 구성
- **Frontend**: React + WebRTC (영상·음성 수집)
- **Backend**: FastAPI 기반 AI 서버
- **AI 모델**
  - Whisper → 음성 STT
  - GPT-4o mini → 답변 평가 및 리포트 생성
  - MediaPipe → 시선·표정 분석
- **DB/Storage**
  - SQLite → 사용자 데이터 저장
  - 영상은 임시 저장 후 자동 삭제
- **Visualization**: JSON → 그래프 변환 및 시각화

## 📅 개발 일정 (6주)
1. **1주차** – 요구사항 정의, UI 설계  
2. **2주차** – 프론트 구축, WebRTC 테스트  
3. **3주차** – 백엔드, Whisper 연동  
4. **4주차** – ChatGPT 리포트 자동화  
5. **5주차** – 시선·태도 인식  
6. **6주차** – 통합 테스트 및 배포 (Vercel·Render)
